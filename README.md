# Noob.typing

## 簡介
你是否有常打錯字導致debug花了半個多小時的困擾？或是被萬能的編輯器寵壞了，離開編輯器就寫不出code來？
那Noob.typing非常適合你！
- 介面簡單，好上手
- 個人化設計，找到自己喜愛的練習環境
- 雙人對戰模式，學習路上不孤獨
- 學習結果分析，了解原因對症下藥

## 網站環境
- Ruby 
- Rails 
- PostgreSQL 

## 網站工具
- 前端：HTML5、CSS、SCSS、JavaScript、Figma、jQuery
- 後端：AJAX、HEROKU、Action Cable、devise
- 版本控制：Git、GitHub
